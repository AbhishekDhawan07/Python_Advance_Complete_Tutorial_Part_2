## 💻 Text-Based Flow 

Start Function  
   ↓  
Check Base Condition?  
   ↓  
 ┌───────────────┐  
 │               │  
Yes             No  
 │               │  
 ↓               ↓  
Return        Recursive Call  
                 ↓  
             Repeat Process 🔁  

---

## ⚡ Simple Explanation

- 🔹 If **base condition is met** → function stops ✅  
- 🔹 Otherwise → function calls itself 🔁  
- 🔹 This continues until it reaches the base case 🎯
